# 🌟 GrameenLight
### Smart Village Pole Monitoring & Electricity Complaint Management System

GrameenLight is an Android application designed to help villages and Panchayats efficiently monitor electrical poles and manage public electricity complaints. Citizens can report issues with photos and GPS location, while administrators can track, update, and resolve complaints in real time.

---

## 📌 Problem Statement

Rural areas often face delays in reporting and resolving electrical faults due to manual complaint systems, poor communication, and lack of centralized monitoring. This project provides a digital platform to improve infrastructure monitoring and complaint management.

---

## 🎯 Project Objective

- Monitor village electrical poles
- Report damaged streetlights and wire faults
- Enable real-time complaint tracking
- Improve communication between citizens and Panchayats
- Support smart village infrastructure management

---

## ✨ Features

### Citizen App
- User Login/Register
- Report electrical issues
- Upload photos
- GPS auto-location capture
- View complaint status
- Interactive village map
- Worker feedback system

### Admin (Panchayat) Panel
- View and filter reports
- Update complaint status
- Monitor village analytics
- Send notifications
- Manage infrastructure data

---

## 🗺 Pole Status Indicators

🟢 Working Pole

## 🏗 Architecture

This project follows **MVVM Architecture**

- Model
- View
- ViewModel

Benefits:
- Better scalability
- Easy maintenance
- Separation of concerns
- Clean code structure

---

## 🔄 Workflow

1. User logs in
2. User selects/report issue
3. GPS and image captured
4. Data uploaded to Firebase
5. Admin receives report
6. Status updated
7. Notification sent to citizen
